## Quick Links

- [Demo](https://next-popcorn-time.vercel.app/)


## Installation

1. Clone repository

   ```
   git clone https://github.com/chrisnph/next-popcorn-time
   ```

2. Install packages

   ```
   cd next-popcorn-time && npm i
   ```

3. Run project locally

   ```
   npm run dev
   ```
